# Measured Frontend

React + TypeScript frontend for the Measured time tracking application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

To build for production:
```bash
npm run build
```

## Technologies

- React 18
- TypeScript
- Vite
- Material UI
- React Router
